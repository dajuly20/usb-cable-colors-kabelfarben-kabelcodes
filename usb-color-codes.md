|----------------|-----------|

| 🟨 Gelb        | USB-C |
| 🟥 Rot         | USB-Mini |
| 🟪 Lila        | USB-Micro |
| 🟥 Rot         | USB-Mini größér als Mini aber langlebiger |

| 🟨 + 🔁        | USB-C Verlängerungskabel (beide Enden gelb, zusätzlich Wiederholungs-Symbol) |
